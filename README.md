# 年金機構 法人番号検索 コピペ有効化

[日本年金機構 事業所検索](https://www.nenkin.go.jp/do/search_section/) の法人番号入力フォームでコピペを有効にするChrome拡張機能。

## 機能

- Ctrl+V によるペースト操作のブロックを解除
- フォームの隣に「クリップボードから入力」ボタンを追加

## インストール

1. このリポジトリをクローンまたはダウンロード
2. Chrome で `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」をオン
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. `extension/` フォルダを選択

## 使い方

1. 法人番号をコピー
2. https://www.nenkin.go.jp/do/search_section/ を開く
3. Ctrl+V で貼り付け、または「クリップボードから入力」ボタンをクリック

「クリップボードから入力」ボタン初回クリック時に、ブラウザからクリップボードへのアクセス許可を求められる場合があります。「許可」を選択してください。

## ファイル構成

```
extension/
├── manifest.json   # 拡張機能の定義
├── early.js        # ページJS実行前にペースト妨害を無効化
└── content.js      # DOM構築後にonpaste属性除去・ボタン追加
```
